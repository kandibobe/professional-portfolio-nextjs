# Оценка стоимости проекта (Project Estimation)

## 1. Справедливая рыночная стоимость

Если бы этот проект заказывали у студии или опытного фрилансера, его стоимость составила бы:

### Базовая разработка (Core): $6,000 - $8,000

- **Стек:** Next.js + TypeScript (максимальная производительность и SEO).
- **Дизайн:** Адаптивная верстка (Mobile First), темная/светлая тема.
- **UI/UX:** Современный интерфейс с анимациями.

### Функциональные модули: $3,000 - $4,500

- **Мультиязычность (i18n):** Полная поддержка Английского и Итальянского.
- **Панель администратора:** Защищенная зона для управления контентом.
- **Портфолио:** Система управления проектами с интеграцией Cloudinary.
- **Формы:** Валидация, защита от спама, email-уведомления.

### Инфраструктура: $1,500 - $2,500

- База данных (PostgreSQL).
- SEO-оптимизация.
- Безопасность (Auth, Rate Limiting).

**ИТОГО РЫНОЧНАЯ ЦЕНА:** $10,500 - $15,000

---

## 2. Предложение для клиента ("Для своих")

Учитывая родственные связи и то, что это первый крупный проект для портфолио:

- **Фиксированная цена:** $1,500 - $2,500 (за готовую техническую часть).
- **Экономия для клиента:** Более 80% от рыночной цены.

---

## 3. Расходы на запуск (Deployment Costs)

### Вариант А: Эконом (Vercel Hobby) - Рекомендуемый для старта

- **Хостинг (Vercel):** $0 / мес.
- **База данных (Neon/Supabase Free):** $0 / мес.
- **Хранилище картинок (Cloudinary Free):** $0 / мес.
- **Домен (.com):** ~$12 / год.
- **Итого:** ~$12 в год.

### Вариант Б: Коммерческий (Vercel Pro) - При росте проекта

- **Хостинг:** $20 / мес.
- **База данных:** $0 - $10 / мес.
- **Домен:** ~$12 / год.
- **Итого:** ~$20-30 в месяц.

---

## 4. План запуска (Checklist)

1.  [ ] **Купить домен** (Namecheap/GoDaddy).
2.  [ ] **Зарегистрироваться на Vercel** (через GitHub).
3.  [ ] **Создать проект в Vercel** (Import Git Repository).
4.  [ ] **Настроить Environment Variables** (Скопировать из .env).
5.  [ ] **Нажать Deploy** и дождаться зеленой галочки.
6.  [ ] **Привязать домен** в настройках Vercel (Domains -> Add).
